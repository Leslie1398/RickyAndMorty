# 🧪 Rick and Morty App - Vue 3

Aplicación creada con Vue 3 + Vite que consume la API de Rick and Morty.

## 🚀 Funcionalidades

- Lista de personajes desde API
- Búsqueda en tiempo real
- Sistema de favoritos
- Modal con información del personaje
- Paginación de personajes
- Pruebas unitarias con Vitest

## 🛠️ Tecnologías

- Vue 3 (Composition API)
- Vite
- Bootstrap 5
- SweetAlert2
- Vitest

## 📦 Instalación

```bash
npm install
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
